# My python project :
Pygame-Platformer-Shooter

## Using the application
* Clone github repository
* Download required: ```$ pip install -r requirements.txt```
* ```$ python main.py```

## Controls
* Move Right - d
* Move Left - a
* Jump - w or Space
* Sprint - Shift
* Aim - Move Mouse
* Shoot - Left Click
* Escape to Menu (if in game) - Escape
* Enter Fullscreen - f
